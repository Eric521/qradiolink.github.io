Building the software

The guide assumes you are using Debian Stretch, but the commands should work on most Linux distributions.

  • Clone the Github repository into a directory of your choice
  • Install software dependencies listed above
  • Compile the protocol buffer sources for your system
  • Run qmake to generate the Makefile
  • Run make (with the optional -j flag)
cd qradiolink
mkdir build
cd ext/
protoc --cpp_out=. Mumble.proto
protoc --cpp_out=. QRadioLink.proto
cd ..
cd build/
qmake-qt4 ..
make
./qradiolink
